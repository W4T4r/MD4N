# Catppuccin Lavender

This directory contains the bundled Noctalia colorscheme data for the Catppuccin Lavender variant.
It is a small asset directory rather than a logic directory: its job is simply to hold the theme files Noctalia expects.

## What Belongs Here

- Colorscheme JSON files used directly by Noctalia
- Small supporting assets that are part of the theme bundle

## What Does Not Belong Here

- General Noctalia configuration
- Plugin code
- Unrelated theme experiments that are not actually used by MD4N
