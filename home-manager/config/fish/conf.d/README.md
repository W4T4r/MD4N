# Fish Conf.d

Shared Fish startup snippets loaded automatically by Fish.
The main repository snippet is [`md4n.fish`](md4n.fish).
It loads the generated local env file at `~/.config/fish/md4n.generated.fish` and then optional machine-local files:

- `~/.config/fish/local.env.fish`
- `~/.config/fish/local.aliases.fish`
- `~/.config/fish/local.functions.fish`
