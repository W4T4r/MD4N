# Fish

Shared Fish shell configuration shipped by the repository.
The whole directory is linked into `~/.config/fish`.
Repository-owned startup behavior lives under [`conf.d/`](conf.d/README.md).
Machine-local MD4N environment values are generated under `~/.config/md4n/fish/`.
Put other machine-local extras under `~/.config/fish/conf.d/local/` instead of this directory.
