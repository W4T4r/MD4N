# Fish

Shared Fish shell configuration shipped by the repository.
The whole directory is linked directly from the repository into `~/.config/fish`.
Repository-owned startup behavior lives under [`conf.d/`](conf.d/README.md).

## Local-Only Fish Files

This linked tree also reserves a few machine-local files that should stay out of Git:

- `md4n.generated.fish`
  generated by `bash scripts/configure-local.sh`; do not edit it by hand
- `local.env.fish`
  local environment variables
- `local.aliases.fish`
  local aliases or abbreviations
- `local.functions.fish`
  local Fish function definitions

[`conf.d/md4n.fish`](conf.d/md4n.fish) sources those files in that order when they exist.
Edit the repository when a change should be shared.
Edit the local-only files above when it should stay on one machine.
