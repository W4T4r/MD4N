# Niri

Shared Niri configuration tracked in Git.
The whole directory is linked directly from the repository into `~/.config/niri`.

## Local-Only Niri Files

This linked tree also allows a few machine-local files to live beside the shared config while staying out of Git:

- `outputs.local.kdl`
  generated by `bash scripts/configure-niri-outputs.sh`
- `browser.sh`
  generated by `bash scripts/configure-local.sh`

Edit the tracked KDL files in this directory when the change should be shared.
Re-run the scripts above when you want to refresh the machine-local files instead of editing the generated output by hand.
