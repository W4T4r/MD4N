# Local Templates

`local/` is runtime-only machine state and is intentionally ignored by Git.
This directory exists to make that boundary obvious.

Use the files here as tracked starter templates for the ignored `local/` tree.
`scripts/configure-local.sh` copies them into `local/` when the corresponding files are missing.

The dependency direction stays one-way:

- `local/flake.nix` imports the shared root flake.
- The shared root flake does not import `local/`.

Runtime layout:

- `local/generated/user.nix`
  Generated by `scripts/configure-local.sh`.
  This is machine-local state and should be regenerated, not hand-edited.
- `local/flake.nix`
  Optional local operational flake.
  Put local-only flake inputs here when they should never appear in the shared root flake.
- `local/nixos/`
  Local NixOS modules such as `hardware.nix`, `swap.nix`, `packages.nix`, and `services.nix`.
- `local/home-manager/`
  Local Home Manager modules such as `packages.nix`, `programs.nix`, `services.nix`, and `fonts.nix`.

Starter files in this directory:

- `flake.nix`
  Starter local flake wrapper with slots for local-only inputs and extra modules.
- `nixos/`
  Starter local NixOS modules.
- `home-manager/`
  Starter local Home Manager modules.

Typical local-only uses:

- Put machine-specific NixOS packages in `local/nixos/packages.nix`.
- Put machine-specific Home Manager programs in `local/home-manager/programs.nix`.
- Put machine-specific Home Manager packages in `local/home-manager/packages.nix`.
- Put local-only flake inputs directly in `local/flake.nix`.

For hand-maintained local overrides, prefer a separate private repository and link the runtime path to it.
That keeps app-managed edits persistent without pushing them into the shared MD4N tree.
A tracked starter lives in [`../private_templates/MD4N-private`](../private_templates/MD4N-private/README.md), and the management workflow is documented in [`../docs/private-repo.md`](../docs/private-repo.md).
If you are unsure whether a change belongs in the shared tree, the ignored local tree, or the private repo, use [`../docs/editing-guide.md`](../docs/editing-guide.md).
