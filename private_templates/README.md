# Private Templates

This directory holds tracked starter content for repositories that are meant to live outside the shared `MD4N` checkout.

Current template:

- `MD4N-private/`
  Starter files for a separate private repository that keeps hand-maintained machine-local state out of the shared tree.

Use these templates as a bootstrap source, not as in-place working directories inside the shared repository.
The recommended `MD4N-private` workflow is documented in [`../docs/private-repo.md`](../docs/private-repo.md).
